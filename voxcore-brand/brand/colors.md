# VoxCore Official Color Palette

- **Primary Brand Color (VoxCore Blue):** #1F6FEB
  - Use for buttons, highlights, charts, CTAs
- **Secondary Accent (Data Cyan):** #2AC3FF
  - Use for hover states, glow effects, active UI elements
- **Dark Platform Background (Core Navy):** #0B1F33
  - Used for hero sections, product UI, dashboards
- **Enterprise Gray:** #6B7280
  - Neutral text and UI
- **Light Background:** #F5F7FA
  - Card backgrounds, light UI
- **Border Gray:** #E5E7EB
  - Card borders, dividers
