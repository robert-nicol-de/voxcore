# VoxCore UI Component Style

## Cards
- border-radius: 8px
- soft shadow
- thin border
- background: white
- border: 1px solid #E5E7EB
- shadow: subtle

## Buttons
- **Primary:**
  - background: #1F6FEB
  - color: white
  - border-radius: 8px
  - padding: 12px 20px
- **Secondary:**
  - outline
  - slight glow on hover
  - darker blue hover

## Icons
- outline style
- thin stroke
- minimal

## Charts
- Primary: VoxCore Blue
- Secondary: Data Cyan
- Neutral: Gray
- Avoid bright rainbow charts
- Use 3–4 colors maximum
