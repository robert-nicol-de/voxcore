## 🎯 Key Concepts

### Emotional Architecture
- **Dark Mode**: Technical, secure, in control
- **Light Mode**: Professional, executive-ready, trustworthy
- **Duality**: Increases perceived maturity

### Token Structure
- **Locked**: Background, text, border colors (never change)
- **Consistent**: Accent and risk colors (same in both modes)
- **Customizable**: Accent, logo, sidebar, risk colors (Phase 2)

### Design Philosophy
- **Professional**: Not cyberpunk, not gaming dashboard
- **Governance-First**: Communicates control and security
- **Accessibility-First**: WCAG AA/AAA compliant

### Implementation
- **Token-Based**: CSS variables, no hard-coded colors
- **Performant**: No re-renders, instant theme swap
- **Persistent**: localStorage saves user preference
