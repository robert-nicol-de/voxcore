# Theme System - Quick Start Guide

## What's New?
VoxQuery now has a complete theme system with Dark, Light, and Custom modes!

## Quick Access
Click the **⚙️ Settings** button in the top-right corner to access theme options.

## Three Theme Options

### 🌙 Dark Mode (Default)
- Professional dark interface
- Easy on the eyes
- Perfect for extended use
- **Status**: ✅ Working

### ☀️ Light Mode
- Clean, bright interface
- High contrast
- Great for daytime use
- **Status**: ✅ Working

### 🎨 Custom Mode
- Create your own theme
- Choose primary color (accents)
- Choose background color
- Choose text color
- Live preview
- **Status**: ✅ Working

## How to Switch Themes

1. Click **⚙️** (Settings) button
2. Select your preferred theme:
   - 🌙 Dark
   - ☀️ Light
   - 🎨 Custom
3. Theme applies instantly
4. Your choice is saved automatically

## Custom Theme Setup

1. Select **🎨 Custom** option
2. Click color pickers:
   - **Primary Color**: Main accent (buttons, links, highlights)
   - **Background Color**: Main background
   - **Text Color**: Primary text color
3. See live preview update
4. Close settings - theme persists

## Color Picker Tips

- Click the color square to open color picker
- Hex value shown next to each color
- Preview box shows how colors work together
- Changes apply instantly

## Persistence

Your theme choice is automatically saved:
- Theme selection persists across sessions
- Custom colors are saved
- No manual save needed
- Works across browser tabs

## Keyboard Shortcuts

- **Tab**: Navigate between options
- **Enter**: Select option
- **Space**: Toggle checkbox

## Mobile Support

✅ Fully responsive on mobile devices
- Touch-friendly color pickers
- Readable on all screen sizes
- Smooth transitions on mobile

## Troubleshooting

**Theme not changing?**
- Refresh the page
- Check browser console for errors
- Clear browser cache

**Colors not saving?**
- Ensure localStorage is enabled
- Check browser storage quota
- Try a different browser

**Preview not updating?**
- Make sure Custom theme is selected
- Try changing a different color
- Refresh the page
