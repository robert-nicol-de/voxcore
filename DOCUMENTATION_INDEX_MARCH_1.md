## 🚀 Quick Start

### To Test Query Execution
```bash
# 1. Open frontend
http://localhost:5173

# 2. Connect to database
Click "Connect" → Select "SQL Server" → Click "Connect"

# 3. Navigate to query view
Click "Ask a Question" on dashboard

# 4. Ask a question
Type: "top 10 customers by sales amount"
Press Enter

# 5. Verify results
Check for results, charts, risk score, execution time
```

### To Understand the System
```bash
# 1. Read visual summary (5 min)
SYSTEM_READY_VISUAL_SUMMARY.md

# 2. Read system status (10 min)
VOXCORE_PLATFORM_STATUS_MARCH_1.md

# 3. Read architecture decisions (10 min)
VOXCORE_ARCHITECTURE_DECISIONS.md
```

### To Deploy to Production
```bash
# 1. Build frontend
npm run build

# 2. Build backend
pip install -r requirements.txt

# 3. Start services
npm run dev
python -m uvicorn voxquery.api:app --host 0.0.0.0 --port 8000

# 4. Verify endpoints
curl http://localhost:8000/api/v1/health
```
